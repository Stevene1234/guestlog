# Guest Log Web App

## Purpose
Provides business guest's with a means to record their time and purpose of visit.

## Goals
- Attractive form-based submission interface.
- Admin interface with filtering.

## Requirements
- Django
- SQLite DB
- Source Code in Git
- Versioning and Changelog
- Markdown Readme
